# Proposta: Minha proposta consiste no desenvolvimento de um site de locação de veículos denominado PUC Localiza, com o objetivo de oferecer aos usuários uma plataforma simples e intuitiva para aluguel e compra de carros, o sistema permite a visualização de veículos disponíveis, incluindo modelos novos e seminovos, organizados de forma clara por meio de um menu lateral de navegação, que facilita o acesso às diferentes seções do site, como aluguel, compra e novidades, além disso, o site conta com uma barra de pesquisa funcional, possibilitando ao usuário encontrar rapidamente veículos de interesse, tornando a navegação mais eficiente e dinâmica, o projeto tem como foco a usabilidade, organização das informações e experiência do usuário, proporcionando uma interface prática, moderna e acessível.
